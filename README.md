# ExpressJS

ExpressJS is a Node Framework .

## Installation

Use the node package manager [npm](https://docs.npmjs.com/) to install framework expressJS.

```bash
npm install 
npm install express

```

## Usage
Client Side

```
npm run start 

```
Server Side
```
npm run start
```
Admin Side
```
npm run
```
